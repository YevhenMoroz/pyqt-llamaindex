# pyqt-llamaindex
Example of use llama-index in Python desktop application

## Requirements
* PyQt5 >= 5.14
* llama-index

## How to Run
1. git clone ~
2. cd pyqt-llamaindex
3. pip install -r requirements.txt
4. cd pyqt_llamaindex
5. python main.py
